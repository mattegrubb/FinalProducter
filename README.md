# InventorySync Pro üì±

InventorySync Pro is an offline inventory management app designed for warehouses with 13,000+ items. It leverages Android SDK, Room Database, Coroutines, CameraX, ML Kit, and Apache POI for high performance and advanced features.

## Features
- **Barcode Scanning**: Quickly scan items using CameraX and ML Kit.
- **Room Database**: Efficiently manage large datasets with SQLite.
- **Excel Export**: Export scanned data to Excel files using Apache POI.
- **Article Import**: Preload and import standard article lists.

## Setup
1. Clone the repository.
2. Open the project in Android Studio.
3. Ensure the following dependencies are installed:
   - Room Database
   - Coroutines
   - CameraX
   - ML Kit
   - Apache POI
4. Place the `ArtikelListaGrund.xlsx` file in the `C:/dev/` directory.

## How to Use
1. Launch the app.
2. Navigate to the main menu to scan items, export data, or import articles.
3. Preloaded articles will be available upon app startup.
